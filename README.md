# Consumer

### Service Provider에 접근하는 모든 클라이언트가 Consumer가 될 수 있다.

* 웹 기반의 Consumer 기술
  * AJAX를 지원하는 모든 기법
  * Web Browser 상에서의 Cross Domain 문제가 발생하기 때문에 이에 대비해야 함.

* 자바 기반의 Consumer 기술
  * HttpClient 클래스(안드로이드), URLConnector 클래스, Spring Framework의 RestTemplate 클래스로 만들 수 있다.
  * Cross Domain 문제 없음.
